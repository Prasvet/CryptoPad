# CryptoPad


Простой инструмент для шифрования и дешифрования текста с GUI.


## Краткое описание


Приложение реализует базовые алгоритмы шифрования:
- инвертирование текста;
- перестановка соседних символов;
- сдвиг символов (шифр Цезаря);
- позиционный сдвиг.

Интерфейс реализован на Tkinter, но архитектура позволяет легко заменить GUI‑библиотеку.

## Запуск


1. Клонируйте репозиторий:
   ```bash
   git clone <ваш-url>
   ```
2. Запустите приложение:
   ```bash
   python main.py
   ```

## Структура

- `core/cipher.py` — алгоритмы шифрования/дешифрования;
- `gui/tkinter_gui.py` — интерфейс на Tkinter;
- `main.py` — точка входа;
- `config.py` — настройки (выбор GUI);
- `tests/test_cipher.py` — юнит‑тесты.


## Основные функции


- **reverse_text** — инвертирует порядок символов;
- **swap_adjacent** — меняет местами соседние символы;
- **shift_by_one** — сдвигает каждый символ на +1 по Unicode;
- **shift_by_position** — сдвиг с учётом позиции символа (по модулю `base`).


## Тестирование


Запустить тесты:
```bash
python -m unittest tests/test_cipher.py
```

## Требования


- Python 3.8+
